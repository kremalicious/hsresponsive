Markup

<div class="container">
  <div class="row">
    <div class="col6 grid2-col1">...</div>
    <div class="col6 grid2-col1">...</div>
  </div>
  <div class="row">
    <div class="col3">...</div>
    <div class="col3">...</div>
    <div class="col3">...</div>
  </div>
</div>
  • .container is always full width
  • .row maximum at 1000px, centered
  • .col* is the number of columns an element is spanning within the grid
  • .grid2-col* controls number of total columns an element is spanning within the optional grid
  • Columns within one .row always need to sum up to 12

The Grid

Grid 12

The main fluid grid system with 12 flexible columns.

In use for viewport sizes 768px and up.

Grid 2

The optional fluid grid system with 2 columns.

In use for viewport sizes between 480px and 768px.

Mix It

The Grid 2 can be used as an addition to Grid 12. All columns of Grid 12 default to full width columns for viewport sizes under 768px. By using the grid2-col1 or grid-col2 classes you can further control the layout between 480px and 768px.

The Grid 12

This grid preview uses both, Grid 12 & Grid 2 (just like the above text blocks). At a viewport size of 768px and down Grid 2 kicks in. Both grids become a 1 column layout for viewport sizes of under 480px.

Natural Box Layout Model

All elements have box-sizing:border-box applied to them in the grid css. This way padding can be applied to all columns/elements without affecting the width or height of a column (or any element).
